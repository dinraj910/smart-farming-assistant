% ================= INTRODUCTION =================
\section{Introduction}
This document lists the initial user stories for the AI-Powered Smart Farming Assistant, written from the perspective of the two identified user classes: \textbf{Farmer} (primary user) and \textbf{Admin} (secondary user). Each story follows the standard format:

\begin{center}
\textit{As a [role], I want [goal], so that [benefit].}
\end{center}

Stories are grouped by feature category to match the Requirement Collection document. Priority follows the same scale: \textbf{High}, \textbf{Medium}, \textbf{Low}. Stories marked \textbf{(Innovation Focus)} correspond to Intercropping Suggestion and Market Analysis, as guided by the project supervisor. The project's geographic focus is \textbf{Kerala}; some stories reflect this directly (e.g. Kerala-relevant crops, Kerala homestead intercropping), while others (e.g. weather, disease-detection training) are intentionally broader for better accuracy, per the scope decision in the Requirement Collection document, Section 3.
